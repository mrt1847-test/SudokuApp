# Play Store 제출 체크리스트 (프로젝트용)

아래 항목들은 코드 외부 설정이므로 릴리즈 전 별도로 확인해야 합니다.

## 스토어 등록 메타데이터
- 앱 이름/간단 설명/자세한 설명
- 앱 아이콘(Adaptive icon 포함)
- 스토어 스크린샷 (휴대폰/태블릿 필요 시)
- 피처 그래픽
- 분류 및 태그

## 정책/법적 요구사항
- 개인정보처리방침 URL
- 데이터 안전 섹션(Data safety) 작성
- 타사 SDK/광고 사용 여부 확인

## 릴리즈 설정
- 서명 키(Upload/Release) 관리
- 버전 코드/버전명 업데이트
- 타겟 API 레벨 및 권한 검토
